Для запуска приложения после клонирования проекта с репозитория и подключения баы данных выполнить следующие команды:

1. composer install
2. php artisan migrate
3. php artisan db:seed
4. php artisan passport:client --personal

Сгенерированная документация по API - https://github.com/PaulYemelyanenko/movies-info-rest-app/blob/master/api-documentation.html

Стек использованных технологий:

 - Apach
 - PHP 7.2
 - MySQL 5.7
 - Laravel 6.0
 - Git
 - IDE PhpStorm
 - composer
 - npm
